# Slider Image Referral Points
ViewPager of images with position reference points

# Installation

    allprojects {
      repositories {
        google()
        jcenter()
        ...
        maven { url 'https://jitpack.io' }
      }
    }

Build.gradle

    dependencies {
      implementation 'com.github.fill93:sliderimage_referral_points:1.0.0'
    }
